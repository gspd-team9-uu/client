Hi! We're a team of 4 students from Uppsala University working on this project for Global Software Product Development. In this project, we build a system for a robot to coordinate with other robots autonomously move goods from one part of a warehouse to another based on temperature and light input.

Our Tech Stack

The robot, the star player, is a Lego Mindstorm EV3 Robot, and was programmed with EV3Python. For the client, we used HTML, CSS, JS and Bootstrap. For the server, RESTful APIs. We used an Arduino/Genuino Mega 2560 with Bluetooth module HC-06 and a temperature/humidity sensor DHT22.
